EXPERIMENT 01

TO PLOT THE GRAPH OF CONTINUOUS SINE WAVE

t=-5:0.001:5;
y=sin(t);
subplot(3,3,1);
plot(t,y);
title('CONTINUOUS SINE WAVE ');
xlabel('Time');
ylabel('Amp');

TO PLOT THE GRAPH CONTINUOUS C0SINE WAVE

t=-5:0.001:5;
y=cos(t);
subplot(3,3,2);
plot(t,y);
title('CONTINUOUS COSINE WAVE ');
xlabel('Time');
ylabel('Amp');

TO PLOT THE GRAPH DISCRETE SINE WAVE

t=-5:1:5;
y=sin(t);
subplot(3,3,3);
stem(t,y);
title('DISCRETE SINE WAVE');
xlabel('Time');
ylabel('Amp');

TO PLOT THE GRAPH DISCRETE COSINE WAVE

t=-5:1:5;
y=cos(t);
subplot(3,3,4);
stem(t,y);
title('DISCRETE COSINE WAVE');
xlabel('Time');
ylabel('Amp');

TO PLOT THE GRAPH UNIT IMPULSE FUNCTION

t=-5:0.002:5;
y=[zeros(1,2500),ones(1,1),zeros(1,2500)];
subplot(3,3,5);
stem(t,y);
title('UNIT IMPULSE FUNCTION');
xlabel('Time');
ylabel('Amp');

TO PLOT THE GRAPH UNIT STEP FUNCTION

t=-5:0.002:5;
y=[zeros(1,2500),(ones(1,2501))];
subplot(3,3,6);
plot(t,y);
title('UNIT STEP FUNCTION');
xlabel('Time');
ylabel('Amp');

TO PLOT THE GRAPH UNIT RAMP FUNCTION

t=-5:0.002:5;
y=t;
subplot(3,3,7);
plot(t,y);
title('UNIT RAMP FUNCTION');
xlabel('Time');
ylabel('Amp');

TO PLOT THE GRAPH PARABOLIC FUNCTION

t=-5:0.002:5;
y=0.5*t.^2;
subplot(3,3,8);
plot(t,y);
title('PARABOLA');
xlabel('Time');
ylabel('Amp');

CONCLUSION:

WE HAVE SUCCESSFULLY STUDIED THE GRAPHS OF CONTINUOS,DISCRETE, TIME SIGNAL ALONG WITH UNIT RAMP,UNIT STEP,UNIT IMPULSE AND PARABOLIC.
